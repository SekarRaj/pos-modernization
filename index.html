Cloud computing is on-demand availability of computer system resources especially data storage (cloud storage) & computing power,
without direct active management by the user. - Wikipedia

and of course economies of scale.

Edge computing brings computation and storage closer to end user

improves response time and bandwidth

Fog Computing - At any point e.g. routers, switches, access points, IoT Gateways

Mobile/Multi Access Edge Computing - Radio access n/w such as GSM, LTE

CloudLet - Typical cloud services near mobile users (Datacenter in a box)

Others Content Delivery N/W, Smart Camera, Pixel Streaming etc.

2. Problems & Solution

Issue faced by Retailers

- Packaged solution from Vendor

- Lack of knowledge on peripherals interaction

- Device variations, Vendor variations

- High Development cost and slow deployment cycle

- No monitoring and diagnostics

- Controlled by Vendor's field engineers

Retailer's Requirement

- No touch installation

- Deployment across various platforms
(Windows/Linux/Android)

- Centralized Monitoring and measurement

- Support across vendors and connectivity protocols

- Offline mode

- Elevated user experience

Technology Stack

- Vertx

- Message Processing using Functional Features

- State machine w/ Java enums

- React with XState for state management

- Deployment using go services
(Plan to convert as K8S Operator)

- Simulator for device specific features
(For Development/Testing Purposes)
